# ⚡️ FastAPI Microservice Template

Легкий и структурированный шаблон для создания микросервисов на **FastAPI**.

### 🛠 Стек технологий
* **FastAPI** — асинхронный фреймворк.
* **SQLAlchemy 2.0** — современная ORM (Async).
* **Alembic** — миграции базы данных.
* **Pydantic v2** — валидация и настройки.
* **uv** — сверхбыстрый менеджер зависимостей.
* **Docker Compose** — быстрая контейнеризация.

### 📂 Архитектура
* **`fastapi-application/`** — изолированное приложение (код + миграции + .env) p.s .env внутри приложения тк рядом в проекте мы можем добавить другой микросервис и у него будет свой .env файл.
* **`api/`** — версионированные эндпоинты (v1, ...).
* **`core/`** — ядро: модели, схемы и конфигурация.
* **`alembic/`** — управление структурой БД внутри пакета(тут же миграции).
